Proteus

This application is designed to manage and organize educational structures within institutions. It supports the following core features:

Organisation Management

Create and manage organisations (e.g., schools, departments). Each organisation can be linked to one or more curriculums to define its educational offerings.

Curriculum Management

Define curriculums that represent structured learning paths. Curriculums can be associated with multiple subjects and linked to one or more organisations.

Subject Management

Manage individual subjects that belong to curriculums. Subjects define the specific units or courses within a curriculum.

Relationship Linking

Link curriculums to organisations
Link subjects to curriculums
